# Skills Configuration

This directory contains AI skills that define coding patterns and best practices for this project.

## Available Skills

### Frontend

- **[nextjs-15.md](nextjs-15.md)** - Next.js 15 App Router patterns
  - File conventions
  - Server/Client components
  - Data fetching
  - Metadata and SEO

- **[react-19.md](react-19.md)** - React 19 patterns with Compiler
  - No manual memoization
  - Server components first
  - New hooks (use)
  - Component organization

- **[tailwind-4.md](tailwind-4.md)** - Tailwind CSS 4 patterns
  - Design tokens usage
  - Common patterns
  - Responsive design
  - Dark mode support

- **[typescript.md](typescript.md)** - TypeScript strict patterns
  - Type definitions
  - Type vs Interface
  - Generics and utility types
  - React-specific types

## How It Works

When Claude Code detects you're working in a specific context (e.g., editing a React component), it automatically reads and applies the relevant skill patterns to its responses.
